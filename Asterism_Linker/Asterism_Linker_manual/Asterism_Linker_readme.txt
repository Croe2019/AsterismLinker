●簡単なルール、操作説明
【ルール】
・プレイヤーである金色の妖精を操作して、ステージ内にあるブロックを破壊して光の種を出現させ、
　光の種が出現している状態でランプに当たることでランプに光を灯すことができます。
  ステージ内にあるすべてのランプに光を灯してステージクリアを目指します。
・ステージごとに設定されている規定ターン数が0なったり、光を灯せていないランプがある状態で
　光の種がなくなってしまうとゲームオーバーです。
・ゲーム内ミッション（やりこみ要素でありゲームクリアに必須の条件ではありません）

・ステージ内にある緑ブロックを全て破壊する。
・すべてのランプの光レベルを最大にする。
・最短ターン数でクリアする

ランプの光レベル一覧
レベル1: 赤
レベル2: オレンジ
レベル3: 黄色

・ランプの光レベルは吸収した光の種の個数によってレベルが変わります
1個:         赤　  　　    レベル1
2個:         オレンジ      レベル2
3個以上: 黄色　　　    レベル3

・プレイヤーの強度とオーラについて

射出キャンセル
　黒の矢印がでてる状態で指を離すと強度調整をやり直すことができます。

青色の矢印 : 青色のオーラを纏い、弱の力でプレイヤーを射出。
　青色の矢印はブロックを破壊できませんが、細かい
　位置調整に使用します。

黄色の矢印 : 黄色のオーラを纏い、中の力でプレイヤーを射出。
　黄色い矢印は緑ブロックと水色のブロックを破壊することができます。

　赤色の矢印 : 赤のオーラを纏い強の力でプレイヤーを射出。
赤い矢印はブロックを破壊できるだけでなく、緑色のブロックを貫通して進むことができます。

【操作方法】
　スワイプでプレイヤーを引っ張り、引っ張った状態で離してプレイヤーを射出することができます。
　補足 : 画面のどこからでもプレイヤーを引っ張ることができます。
　
●ブロックの種類
　緑ブロック : 1回の衝突で破壊することができます。
　赤ブロック : プレイヤーが赤色の矢印の状態で2回衝突することで、破壊することができます。
　水色のブロック : 性質は緑ブロックと同じです。

●プレイヤー射出時のパワー減衰について
プレイヤーを射出すると徐々にスピードが減少し、
最終的に停止してターンが進みます。
例として、赤の最大パワーで射出したときは、
はじめは赤のオーラを纏い、
緑ブロック貫通性能がありますが、スピードが減衰して黄色のオーラに変わったとき、
緑ブロックは破壊はできるが、貫通は不可…といったものです。

●光の種の性質
　1ターンごとに小さくなっていき、3ターン経過すると光の種が消えてしまいます。
